# Unity-Dialog-and-Localization-Tool
Dialog and Localization Tools made with Unity for UWO cs4482

## Description
In the game, click the pause button on the top-right or press Esc to change languages

To add dialog texts, use localization tool, which is the scriptable object Assets/Tools/Languages/Localization.asset

![Image](https://github.com/EmperorGesar/Unity-Dialog-and-Localization-Tool/blob/master/screenshot3.png)

To edit dialog, use dialog tool, which is the scriptable object Assets/Tools/Dialog/Dialog.asset

- Click "Edit graph" to edit

- Right-click and select "Question And Answer" to create a new node

- Drag "Prev" and "Next" ports to modify the order

- Assign the names of dialog text keys in the localization tool

The dialog tool is made with [xNode](https://github.com/Siccity/xNode)

![Image](https://github.com/EmperorGesar/Unity-Dialog-and-Localization-Tool/blob/master/screenshot4.png)
